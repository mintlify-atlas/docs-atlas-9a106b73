Build AI agent teams that decompose goals into tasks automatically. Define agents with roles and tools, describe a goal — the framework plans the task graph, schedules dependencies, and runs everything in parallel.

3 runtime dependencies. One `runTeam()` call from goal to result.

## Why Open Multi-Agent?

<AccordionGroup>
  <Accordion title="Auto task decomposition">
    Describe a goal in plain text. A built-in coordinator agent breaks it into a task DAG with dependencies and assignees — no manual orchestration needed. Tasks that don't depend on each other run in parallel automatically.
  </Accordion>
  <Accordion title="Model agnostic">
    Claude, GPT, GitHub Copilot, and local models (Ollama, vLLM, LM Studio) can work together in the same team. Set a `baseURL` to point any agent at any OpenAI-compatible endpoint.
  </Accordion>
  <Accordion title="In-process execution">
    No subprocess overhead. Everything runs in one Node.js process. Deploy to serverless, Docker, or CI/CD pipelines without any infrastructure dependencies.
  </Accordion>
  <Accordion title="Built-in tools">
    Five production-ready tools ship out of the box: `bash`, `file_read`, `file_write`, `file_edit`, and `grep`. Define custom tools with a Zod schema in ~10 lines of code.
  </Accordion>
</AccordionGroup>

## Get started

<Steps>
  <Step title="Install">
    ```bash
    npm install @jackchen_me/open-multi-agent
    ```
  </Step>
  <Step title="Set your API key">
    ```bash
    export ANTHROPIC_API_KEY=your-key-here
    ```
  </Step>
  <Step title="Run your first team">
    ```typescript
    import { OpenMultiAgent } from '@jackchen_me/open-multi-agent'

    const orchestrator = new OpenMultiAgent({ defaultModel: 'claude-sonnet-4-6' })

    const team = orchestrator.createTeam('my-team', {
      name: 'my-team',
      agents: [
        { name: 'researcher', model: 'claude-sonnet-4-6', systemPrompt: 'You research topics.' },
        { name: 'writer', model: 'claude-sonnet-4-6', systemPrompt: 'You write clear summaries.' },
      ],
      sharedMemory: true,
    })

    const result = await orchestrator.runTeam(team, 'Summarize the key benefits of TypeScript generics.')
    console.log(result.agentResults.get('coordinator')?.output)
    ```
  </Step>
</Steps>

<Note>
  Requires Node.js 18 or higher. See the [quickstart](/quickstart) for a complete walkthrough including all supported providers.
</Note>
